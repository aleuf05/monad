# Draft — libfive pipeline status

libfive supports a headless pipeline, but not as a standalone CLI product.
The documented path is to build a custom application on top of the kernel and
bindings, while Studio remains the GUI layer.

On this host the required build dependencies are not yet installed, and package
installation is blocked in-session. The follow-up has been recorded in the work
queue and a validation report has been written.
